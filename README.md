## Installation

Clone the repo:

    git clone https://github.com/agnkos/stack-tags.git

Install npm packages:

    npm install 

Or:

    npm ci

Run the app:

    npm start

Run Storybook:

    npm run storybook

## Technologies

Built with: React.js, TypeScript, TanStack Query, Axios, Material UI, Storybook, Vite
